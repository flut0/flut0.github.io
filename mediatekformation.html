01Contexte

Mediatekformation est une application web permettant d'accéder à des vidéos d'auto-formation proposées par une chaîne de médiathèques, également accessibles sur YouTube. Le projet d'origine, fourni par l'enseignant, ne comprenait qu'un front office basique. L'atelier consistait à améliorer et compléter l'application en 4 missions.

Dépôt d'origine : github.com/CNED-SLAM/mediatekformation

03Missions réalisées
Mission 1

Qualité du code

Analyse et correction des problèmes détectés par SonarLint. Ajout d'une fonctionnalité de comptage.

  • Remplacement des chaînes littérales par des constantes
  • Correction des blocs manquants et conditions imbriquées
  • Accessibilité : attributs alt et summary
  • Colonne "nb formations" avec tri dans les playlists
Mission 2

Back office admin

Développement d'une interface d'administration complète avec authentification sécurisée.

  • CRUD formations, playlists, catégories
  • Authentification Symfony Security (ROLE_ADMIN)
  • Redirection automatique vers /login
  • Règles métier : suppression conditionnelle
Mission 3

Tests PHPUnit

Suite de tests automatisés couvrant les différentes couches de l'application.

  • Tests unitaires (entité Formation)
  • Tests de validation (contrainte date)
  • Tests d'intégration (Repository)
  • Tests fonctionnels (WebTestCase)
  • Résultat : OK (16 tests, 21 assertions)
Mission 4

Déploiement & CI/CD

Mise en production avec automatisation du déploiement et des sauvegardes.

  • Déploiement sur Alwaysdata (SFTP + SSH)
  • Sauvegarde automatique BDD quotidienne à 02h00
  • GitHub Actions : déploiement continu sur push
