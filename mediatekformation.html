Mediatekformation

Application web de vidéos d'auto-formation développée avec Symfony 6.4. Ajout d'un back office d'administration, de tests PHPUnit et déploiement en ligne avec CI/CD.

Contexte

Le projet Mediatekformation est une application web permettant d'accéder à des vidéos d'auto-formation proposées par une chaîne de médiathèques. Le projet d'origine, fourni par l'enseignant, ne contenait qu'un front office. L'atelier consistait à réaliser 4 missions pour améliorer et compléter l'application.

Dépôt d'origine : github.com/CNED-SLAM/mediatekformation

Langages et technologies

PHP 8.2 Symfony 6.4 Twig Doctrine ORM MySQL PHPUnit Bootstrap 5 Git / GitHub GitHub Actions Alwaysdata SonarLint Composer

Missions réalisées

Mission 1

Qualité du code et nouvelles fonctionnalités

Corrections SonarLint (constantes, blocs manquants, accessibilité). Ajout d'un compteur de formations par playlist avec tri.

Mission 2

Back office d'administration

CRUD complet sur formations, playlists et catégories. Authentification sécurisée avec Symfony Security (ROLE_ADMIN).

Mission 3

Tests PHPUnit

16 tests automatisés : unitaires, validation, intégration (Repository) et fonctionnels (WebTestCase). Résultat : OK (16 tests, 21 assertions).

Mission 4

Déploiement et exploitation

Déploiement sur Alwaysdata, sauvegarde automatique quotidienne de la BDD, déploiement continu via GitHub Actions.
